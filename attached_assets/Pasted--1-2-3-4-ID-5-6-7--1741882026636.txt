Дата обработки зкрашывались похожыми цветами, которые ты зкрашывал на уровнях разработчиков, в Локация: цвета будут менятся в зависимости от локации, если локация москва: тогда будет цвет: тёмный с оттенком фиолетового, если Санкт-Петербург тогда цвет будет тёмный с оттенком жёлтого, и если США тогда будет цвет тёмный с оттенком зелёного, если Барнаул тогда будет цвет более яркий цвет, с оттенком жёлтого, и зелёного, если Алматы тёмный с оттенком синего, если Армения тогда будет цвет тёмный с оттенком красного, если Минск тогда будет цвет тёмный с оттенком зелёного, и синего, если Екатеринбург тогда цвет будет тёмный с оттенком фиолетового, и синего, если Белгород тогда будет цвет тёмный с оттенком красного, и зелёного, если Тбилиси тогда будет тёмный с оттенком жёлтого и красного если Сербия тогда будет цвет тёмный с оттенком синего, и жёлтого, если Астана тогда будет тёмный с оттенком розового, если Всеволожск тогда будет тёмный с оттенком жёлтого и розового. В названии цвет будет менятся в зависимости от того, какая длинна в названии, если длинна будет больше 30 символов, тогда будет цвет тёмный с оттенком фиолетового, если будет меньше 30 символов, тогда будет цвет тёмный с оттенком жёлтого, если будет меньше 5 символов тогда цвет будет тёмный с оттенком зелёного. А в компании цвет будет менятся в зависимости тоже от длинны, ID цвет будет менятся в зависимости от того, например: если пятаое число в ID например 5, а последнее число будет анпример 5 тогда цвет будет рандомным, но только выбранный цвет, под нашы цвета, которые есть в таблице. А описание будет менятся в зависимости от того, например: если в описании упоминаются языки програмированния, тогда цвет будет тёмно-фиолетовый, если же не упоминаются, тогда цвет будет тёмно жёлтым, если упоминается в описании уровень разработчика: Junior: цвет тёмный с оттенком синего, Middle: цвет тёмный с оттенком красного, Senior: цвет тёмный с оттенком розового. Зарплата от/до цвет будет менятся в зависимости от того, например: зарплата в RUB: если будет меньше 2тысяч рублей, тогда цвет будет тёмено розовый, если будет больше 2тысяч рублей, тогда цвет будет тёмно жёлтый, если будет больше 100тысяч рублей, тогда цвет будет тёмно синий.
